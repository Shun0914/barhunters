# Figma Make 用プロンプト（部門長向け「人的資本ポイント」MVP UI）

以下を **Figma Make のプロンプト欄にそのままコピペ**して使ってください。  
目的：**日本語UI**で、西部ガスHDのクエスト文脈に沿った **スマホ向けMVP（2画面）** を生成・調整すること。

---

## プロンプト本文（ここからコピー）

```
あなたはプロダクトデザイナーです。以下の要件で、モバイルファーストのUIをFigma上に作成・整えてください。

## 背景
- クライアントはインフラ系大企業（都市ガスグループ）の人事・人的資本施策チーム。
- 経営から「人への投資の効果を説明できるように」と求められている一方、本音の目的は「数字に出ない努力や挑戦を、部門長が拾いやすくし、職場の温度（ワクワク・称賛）を上げること」に近い。
- ポイントは「人事評価には使わない／見える化専用」という前提を画面でも明確にしたい（不信感・やらされ感を減らす）。

## 作ってほしいもの（MVP・2画面）
ビューポートはスマホ想定（幅390px前後）。トーンは「堅すぎないが、インフラ企業の現場でも使える信頼感」。角丸・余白はゆったり。C向けアプリのような“ふわっと楽しい”寄りでもよいが、**監視・評価ツールに見えない**こと。

### 画面1：ホーム（部門長ダッシュボード）
- アプリ名（仮）：「きらめきノート」
- サブコピー例：「数字に出にくい頑張りを、いま拾う」
- 画面上部に**注意書きバナー**（短く）：本機能で付与したポイントは人事評価には使用しません（見える化専用）。
- **今月の配分ポイント残**を大きく表示（例：320 / 500 pt）。補足の小さな説明：例「メンバー数×10ptが今月の配分バジェット」など。
- **最近拾った記録**リスト（3件程度のカード）。各行に「付与pt」「氏名（仮）」「一言」「カテゴリ」が読めること。
- 下部に主CTA：**「記録する」**（フルワイドのボタン or FABでも可）

### 画面2：記録フォーム（低摩擦入力）
- 戻る導線（「戻る」）
- 見出し：「記録する」
- **カテゴリ**はチップ選択（複数行折り返し可）。ラベル例：
  - 助け合い
  - 小さな試行
  - 1on1
  - 残業時間の削減
  - 資格・学び
  - チャレンジ／うまくいかなかったこと
- **対象メンバー**：検索フィールド（プレースホルダ「名前で検索」）
- **できごと（1行）**：1行テキスト。プレースホルダ例「例：引き継ぎをチェックリスト化して短縮した」
- **付与ポイント**：プリセット（例 5 / 8 / 10）＋必要なら数値調整UIの余地を示す
- 主CTA：**「保存」**

## コピー（日本語）のルール
- すべて**日本語**で。英語ラベルは使わない。
- 「評価」「査定」「監視」に連想される言い回しは避け、「見える化」「称賛」「拾う」「記録」など中立〜前向きの語を使う。
- インフラ現場（工事・生産）の部門長でも選びやすいよう、「挑戦」だけに寄せず**小さな行動**のカテゴリを並べる。

## デザインの方向性
- カラーは落ち着いたティール／ネイビー系のアクセント＋オフホワイト背景が好ましい（ガス会社っぽすぎるイラストは必須ではない）。
- アクセシビリティ：コントラストは十分。タップ領域は44px相当を意識。
- コンポーネント化：ボタン、チップ、リスト行、カード、注意バナーを再利用可能に。

## 成果物
- 2画面を同じフレーム内に並べ、**フローが一目で分かる**ようにしてください。
- 主要な余白・タイポのルールを簡単にメモ（コメント）で残してください。
```

## メモ（チーム用）

- **管理（admin）ダッシュボード**：経営・人事向けに **`/admin`** で **デスクトップ幅**の「きらめきノート - 部門長ダッシュボード」を追加する場合の追記例（プロンプト末尾に貼る）：
  - 「ルート `/admin` に、幅1280px想定の管理画面。ヘッダーにタイトル・最終更新。上段にKPI4枚（エンゲージメント・離職率・平均残業・人的投資pt）。中段は2:1で左に『人的投資と経営指標の推移』折れ線エリア、『投資カテゴリ別の分布』（円＋横棒）、右に『経営へのインパクト』グラデカード（換算額・ROI示意）、『気づきとアラート』。下段に定性ストーリー一覧とメンバー散布図。モバイル下部タブからは開かず、別URL扱い。」
- 既存の英語ベースのFigmaファイル（MCP生成）がある場合は、Makeに **「この画面構成を日本語コピーに全面差し替え」** と追記してもよいです。
- 長谷さんヒアリングで出た **「ガス坊やを使ってよい」** を入れたい場合は、プロンプト末尾に1行追加：  
  `マスコット「ガス坊や」をヘッダーや空状態イラストに軽く使ってよい（過剰に子供向けにならない程度）。`
