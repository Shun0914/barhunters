## 1. 顧客起点の1枚図（仮説版）を固定する

| | 内容 |
|---|------|
| **やること** | `仮説ワーク_顧客→行動→指標_テンプレート.md` の **V1〜V5 表**を、FigJam「しゅんすけ」エリアと**同じ列**で**3〜5行に圧縮**（統合・削除のメモ付き）。**顧客→行動→先行→遅行**の一文ストーリーを口頭30秒で言える版を書く。 |

## 3. 4/21 正式提案の「スコープ宣言」を書く

| | 内容 |
|---|------|
| **やること** | **含める**／**含めない**を表形式で各1列（例: MVP画面の範囲、連携するマスタ、評価との距離の文言、試算の深さ）。**最小・標準・理想**の3段があれば議論しやすい（理想は「今回やらない」に落とす候補）。 |

| **今回やらないこと** | 「6月全社の運用詳細まで全部確定」と言い切ること（**承認後の設計フェーズ**に逃がす）。 |
