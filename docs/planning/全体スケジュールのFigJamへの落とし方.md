# 全体スケジュールを FigJam に落とす方法

## 方法1：MCP でガントチャートを生成（今回やったやり方）

1. Cursor で「FigJam に全体スケジュールを落としたい」と依頼する
2. MCP の **generate_diagram** で Mermaid の **gantt** を生成する
3. 返ってきた **Claim URL**（`claimFileUrl`）を開く  
   - 例: `https://www.figma.com/online-whiteboard/create-diagram/...?utm_content=edit_in_figjam`
4. ブラウザで開くと「FigJam に追加」のようなオプションが出るので、**既存の FigJam ファイルに追加** または **新規 FigJam で開く** を選ぶ
5. ガントチャートが FigJam のボードに挿入される

**注意**: Claim URL は有効期限（例: 1週間）があるので、期限内に開いて FigJam に取り込む。

---

## 方法2：手動で FigJam に作る

### A. スケジュール表をそのまま貼る

- リポジトリの `docs/research/Tech0_BtoC_Bootcamp_Step4_西部ガス様企業クエスト.md` の「スケジュール」セクションの表をコピー
- FigJam で **テキスト** または **表（Table）** を置いて貼り付け

### B. タイムライン／スティッキーで可視化

1. FigJam で **横長のエリア** を 1つ用意（タイムライン用）
2. 上に「2月」「3月」「4月」「5月」など **月ラベル** を並べる
3. 各フェーズ（キックオフ / Q1 / 壁打ち① / Q2 / 壁打ち② / Q3 / ブラッシュアップ / 最終発表）を **スティッキー** で作成
4. 該当する月の下にスティッキーを並べ、必要なら矢印やコメントでつなぐ

### C. FigJam の「スケジュール」テンプレートを使う

1. FigJam で **新規ボード** → テンプレートから「**プロジェクト計画**」「**タイムライン**」などを検索
2. テンプレの期間・タスクを、クエストのフェーズ名・日付に差し替える

---

## 方法3：Mermaid を自分で編集してから MCP で生成し直す

- ガントの内容を変えたい場合は、**Mermaid の gantt 構文** を編集してから、再度「この Mermaid で FigJam に図を生成して」と依頼できる
- 例: 担当者ラベルを足す、マイルストーンだけにする、など

## 今回生成したガントの構成（参考）

| セクション   | タスク概要                         |
|-------------|------------------------------------|
| キックオフ   | クエスト説明・チーム編成           |
| QUEST1 知る | 事例研究・戦略骨子の策定           |
| 壁打ち      | 今長谷氏ヒアリング・方向性確認     |
| QUEST2 描く | 推進シナリオ設計・効果試算         |
| 壁打ち      | 中間発表・フィードバック・軌道修正 |
| QUEST3 創る | 3層設計・MVP開発                   |
| FINAL       | 全成果物統合・プレゼン準備 → 最終発表 5/20 |

必要なら、この構成をベースに Mermaid を書き換えて再生成できる。
