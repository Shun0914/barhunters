# 週次「今週のやること」

講義は**水曜**。1週間サイクルで、その週の宿題・マイルストーンをまとめたファイルをここに置く。

---

## ファイルのきまり

- **ファイル名**: `YYYY-MM-DD_week.md`（その週の**水曜日**の日付）
- **中身**: その週の「今週のやること」（宿題＋必要ならクエストのマイルストーン）をリスト形式で記載。

例: `2026-03-11_week.md` → 3/11（水）〜次の火曜までの週のやること。

---

## 今週の宿題への参照

- **一覧**: 親ディレクトリの [homework_backlog.md](../homework_backlog.md) を参照。
- **今週のやること**: 直近の水曜日付の `YYYY-MM-DD_week.md` を開く（例: [2026-03-11_week.md](2026-03-11_week.md)）。

## FigJamとの同期

各週ファイルの内容を、FigJamの「**今週のやること**」エリアにコピペして使う。更新時は「FigJam用コピペ用」テキストをチャットで受け取り、ボードに貼る。
