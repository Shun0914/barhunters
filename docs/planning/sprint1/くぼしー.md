## 課題③：西部ガスホールディングス株式会社 企業調査まとめ

### ① 企業は誰に何を提供しているか

西部ガスHDは、都市ガスの製造・供給・販売を中核に、LPG販売、電力販売・再生可能エネルギー、不動産、食品・介護・情報処理などの多角的なサービスを展開するグループ持株会社です。Saibugas 事業の対象は大きく2つに分かれます。

- **個人（家庭・消費者）**：都市ガス・電力・LPG供給、住宅・不動産、食品・介護サービス
- **法人（企業・自治体）**：エネルギー供給、熱供給、LNG販売、太陽光・風力発電、情報処理サービス

### ② 主な顧客や利用者は誰か

福岡市・北九州市を主要地盤とし、都市ガス販売量で全国5位の規模を持つ地域インフラ企業です。九州北部（福岡・北九州・熊本・長崎・佐世保）を中心とした一般家庭・企業・自治体が主な顧客であり、お客さま戸数は114万1,000戸（2025年3月時点）にのぼります。

### ③ 企業の強み・特徴は何か

- **地域密着の導管インフラ網**  
  導管とは地下に張り巡らされたガスのパイプのことで、LNGタンクで作られたガスを各家庭・工場に届ける仕組みです。一度埋設すると数十年使える一方で、新規参入者が独自に構築することは現実的に不可能であり、強固な参入障壁として機能します。日本全体でも都市ガスの供給エリアは国土の約6%に限られており、導管が通っていない地域ではガス会社を選ぶこと自体ができません。西部ガスHDの場合、玄界灘沿いの供給エリアが地理的に連続しており、ひびきLNG基地への拠点集約にも繋がるという追加の強みがあります。

- **多角化・地域共創**  
  地域に根差したネットワークで不動産・食関連・介護など多様な事業を展開しつつ、外部パートナーのアイデアや技術と掛け合わせた新規事業創出にも取り組んでいます。

- **高い顧客信頼**  
  ガス小売部門においてJCSI（日本版顧客満足度指数）調査で4年連続No.1を獲得しており、3年後平均定着率も95.8%と高水準を維持しています。

### ④ 業界・市場の中で起きている変化

- **変化① 自由化による競争激化**  
  2017年に都市ガス小売事業が全面自由化され、電力会社・通信会社・LPガス会社など異業種からの参入が進み、2025年4月時点でガス小売事業者は283社にのぼります。ただし現時点ではインフラの壁が高く、競争はまだ限定的であり、2024年3月時点でシェア5%以上の競争者が2者以上存在する区域は存在しません。

- **変化② エネルギー情勢の不安定化**  
  LNGをはじめとする燃料価格は2022年夏頃に高騰し、その後2023年以降は下落に転じましたが、地政学リスクによるボラティリティは依然として高い状態が続いています。原料調達コストの変動が経営の安定を直接脅かすリスクとなっています。

- **変化③ 脱炭素・カーボンニュートラルの加速**  
  政府は2030年度にガス供給量の5%をカーボンニュートラル化するという目標を掲げており、合成メタン（e-メタン）やバイオガスの導管注入に向けた制度整備が進んでいます。東京ガス・大阪ガス（Daigasグループ）・西部ガスHDなど各社が再エネ開発やe-メタン実証事業を競うように推進しています。

- **変化④ ガスと電力の業界の壁が崩れている**  
  電力自由化との相乗効果でガス・電気のセット販売が活発化し、東京電力がガス市場に参入する一方、ガス会社が電力事業に参入するという相互侵食が進んでいます。業界各社が「エネルギー総合サービス会社」への変身を目指しており、西部ガスHDもひびき発電所（2026年3月運開予定）の稼働によって電力事業の本格拡大を図ります。

- **変化⑤ 需要の構造的減少**  
  工業用ガス需要は減少傾向にあり、家庭用・商業用は微増・横ばいが続いています。人口減少・少子高齢化・省エネ機器の普及・オール電化の拡大が重なり、「黙っていても需要が伸びる時代」はすでに終わっています。

### ⑤ 今回の企業テーマはなぜ出てきたか

今回のクエストテーマが出てきた背景として、以下の3層の文脈が重なっています。

- **外部圧力：法的・開示義務の強まり**  
  2023年から有価証券報告書への人的資本情報の開示義務化が始まり、定性情報だけでなく定量指標の開示が求められる時代になりました。ESG評価の観点からも人的資本への取り組みは投資家からの注目度が高まっており、「見せる経営」が求められています。

- **経営課題：事業変革に人材投資が不可欠**  
  脱炭素・DX・多角化という3つの変革を同時に進めるには、従来の「ガスを安全に供給する人材」から「新しい価値を創れる人材」への転換が必要です。西部ガスHDの中期経営計画ACT2027では人的資本経営を全戦略を支える基盤として位置づけており、従業員エンゲージメントスコア65%以上、女性管理職比率6%など具体的なKPIを設定しています。

- **内部課題：投資効果の「見えなさ」問題**  
  人件費を引き上げながら「その効果を経営として証明できない」という課題は、経営会議での予算承認や施策継続の判断を困難にします。クエストの背景にある「他のガス会社にも聞いたが、解決策は誰も持っていない」という言葉が示すように、これは業界全体の未解決問題です。人的資本ROIは現時点で2022年度34%・2023年度31%・2024年度34%と推移していますが、これがどの施策によって生み出されたのかを証明する仕組みがないのが現状です。

## 課題④：観点・論点・情報不足・今後の動き

### どの観点で分解したか

**Why / What / How の3軸**で分解しました。この順番はそのまま最終プレゼンのストーリー構成（なぜ必要か→何を測るか→どう動かすか）にもなります。そこに今回の対話で出てきたあなた自身の疑問を各軸に紐付けています。

### 最も重要だと考える論点（3つ）

1. **① 「結果を出せ」の結果とは何か（Why）**  
   これが最重要です。経営層が求める「結果」と人事部門が考える「結果」にズレがあると、どれだけ良い仕組みを作っても経営会議を通せません。ここを定義しないまま進むとプロジェクト全体がズレます。

2. **② 人事施策は利益にどう繋がるか（What）**  
   1P＝10万円という換算ロジックの土台になる論点です。この繋がりが説明できないと、ポイント集計しても「だから何？」で終わります。

3. **③ 1P＝10万円の換算根拠は何か（What）**  
   中期経営計画に明記されている数字ですが、誰がどういうロジックで決めたかが不明です。この根拠が曖昧なまま6,000P達成を報告しても、経営層を動かす力を持ちません。

### 現時点で情報不足な論点（3つ）

1. **① 現状どんなデータを保有しているか（How）**  
   タレントマネジメントシステムの整備状況、人事データの一元管理の現状が不明です。システム設計の前提となるため、ヒアリングで最初に確認すべき点です。

2. **② データはどこから収集するのか（How）**  
   ポイントを誰がどのタイミングで入力・申請するのか、既存の業務フローとどう接続するのかが見えていません。

3. **③ 他社に測定事例は本当にないのか（Why）**  
   「他のガス会社にも聞いたが誰も持っていない」という発言は事実かもしれませんが、ガス業界以外・海外事例まで含めると状況が変わる可能性があります。ここは学習フェーズで潰せます。

## 仮説（課題⑤）

### 仮説① システムのゴール設定

| 項目 | 内容 |
|------|------|
| **仮説本文** | もし今回のシステムのゴールが「ポイントの集計・可視化」ではなく「エンゲージメント向上と発揮成果の最大化」に設定されているなら、システムの設計は単なる申請・承認フローにとどまらず、従業員の行動変容を促す仕掛けを組み込む必要があるはずだ。 |
| **誰についての仮説か** | 西部ガスHDの人財戦略部・経営層 |
| **この仮説を置く理由** | ACT2027のROICツリーに「エンゲージメント向上と発揮成果の最大化」が価値創造基盤の強化として明記されており、人的資本投資の最終目的はポイント数値の達成ではなく企業価値向上に繋がる行動変容であると読み取れるため。 |
| **正しければ起こること** | システムに「どんな行動がエンゲージメント向上に繋がるか」を従業員が実感できる仕掛けが必要になる。ポイントを貯めること自体が目的化しないよう、行動と効果の因果を見える化する設計が求められる。 |
| **誤っていれば起こること** | ポイントの集計・承認・可視化が主目的であり、申請フローと経営ダッシュボードの実装がゴールになる。システムの複雑さは大幅に下がる。 |
| **何を確認できれば前進できるか** | ヒアリングで「このシステムを使って従業員にどう変わってほしいか」を今長谷氏に直接聞く。 |

### 仮説② 因果連鎖の設計

| 項目 | 内容 |
|------|------|
| **仮説本文** | もし換算対象に「業務改善・コスト削減」などの直接効果だけでなく「エンゲージメント向上・離職率低下」などの間接効果も含まれており、かつその接続が未設計であるなら、間接効果をROIC改善に繋げる換算ロジックを定義することがプロジェクトの核心であり、日本企業全体で1%しか実現できていない領域に挑むことになるはずだ。 |
| **誰についての仮説か** | 西部ガスHDの全従業員・部門長・経営層 |
| **この仮説を置く理由** | ACT2027のROICツリーに因果連鎖の骨格は示されているが、「発揮成果の最大化→ROIC改善」の定量的な接続は未設計であることが資料から確認できた。また日本総研の調査で、企業体の収益率と人的資本投資を完全に連動した形で可視化している企業は全体の1%にとどまることが実証されており、この難しさは業界全体の未解決課題である。 |
| **正しければ起こること** | 間接効果を含めた換算ロジックの設計がプロジェクトの最大の成果物になる。具体的には「この施策→エンゲージメント向上→離職率低下→採用コスト削減→X万円」という連鎖を定義できれば、1P＝10万円の根拠が初めて成立する。 |
| **誤っていれば起こること** | 直接効果のみが対象であれば換算は比較的シンプルになる。業務改善提案1件の経済効果を積み上げる計算で6,000P達成のロジックが成立する。 |
| **何を確認できれば前進できるか** | ヒアリングで以下の2点を確認する。「人的資本投資が利益に繋がった」と社内で実感できた具体的な事例が1件でもあるか。人的資本ROI（現状34%）がなぜこの数値なのか、社内でどう解釈されているか。 |

### 仮説③ 目標設定の妥当性

| 項目 | 内容 |
|------|------|
| **仮説本文** | もし6,000P・10,000Pという目標数値に積算根拠がないなら、現場の部門長が「何をすればいいかわからない」状態で2026年6月の全社運用が始まり、制度が形骸化するリスクが極めて高いはずだ。 |
| **誰についての仮説か** | 西部ガスHDの部門長・現場従業員・人財戦略部 |
| **この仮説を置く理由** | 中期経営計画に2026年度6,000P・2027年度10,000Pと明記されているが積算根拠が資料に示されていない。エンゲージメント調査が2025年度開始であることを踏まえると、6,000P達成時点でまだ1年分のデータしかなく目標達成と業績改善の因果を証明するには期間が短すぎる可能性がある。さらに業界全体で同様の取り組みを実現している企業が1%にとどまることから、根拠なき目標設定である可能性が高い。 |
| **正しければ起こること** | 目標の積算根拠の設計がプロジェクトの重要アウトプットになる。全社6,000Pを部門数・従業員数で割り部門別目標を配分し、1ポイント獲得に必要な取り組みの種類と件数を定義し、達成可能かのシミュレーションを作成する必要がある。 |
| **誤っていれば起こること** | すでに部門別の積算根拠と取り組みカテゴリが存在しており、このプロジェクトの役割はシステム実装に絞られる。 |
| **何を確認できれば前進できるか** | ヒアリングで以下の3点を確認する。「6,000Pはどういう根拠で設定したか」を直接聞く。根拠がなければ「一緒に設計する」という提案ができる。西部ガスHDの全部署数・部門数・従業員数を確認し部門別配分のシミュレーションを計算できる状態にする。2026年6月の全社運用開始までに現場の部門長への説明・研修の機会が設けられているかを確認する。 |
