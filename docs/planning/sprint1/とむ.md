## 課題③：企業調査

### ① 企業は誰に何を提供しているか

九州を主要地盤とする都市ガス大手（販売量全国5位、東証PRM上場）。家庭・法人・産業向けに都市ガス・LPG・電力・再生可能エネルギーを供給。2021年に持株会社体制へ移行し、連結子会社46社でガス以外にも不動産・食関連・温浴・遊園地・情報システム事業を展開する「九州の総合インフラ企業」へ転換中。

### ② 主な顧客や利用者は誰か

- **外部顧客**：一般家庭（都市ガス・LPG契約者）／工場・商業施設・法人（エネルギー供給）／地域住民（温浴・飲食・遊園地などサービス事業）
- **今回のクエストにおける内部ユーザー**：現場社員（貢献をポイントで申告する側）／部門長（申請を承認する側）／経営層／人財戦略部（ダッシュボードで効果を確認する側）

### ③ その企業の強み・特徴は何か

- 福岡市・北九州市を主要地盤とした九州圧倒的ネットワーク
- ひびきLNG基地という国際エネルギー拠点を保有
- 「ソウゾウ大学」（2022年開校）など独自の人財育成プログラムを先行実施
- タレントパレット導入など人財の見える化への積極投資
- 2025年11月に業界初となる「人的資本レポート2025」を発行 → 投資→従業員成長→企業価値→地域還元の「インパクトパス」を可視化
- 人的資本委員会を新設しグループ会社社長が委員として参画（経営レベルで人的資本経営を推進する体制が今年整った）

### ④ 業界や市場の中で、どのような変化が起きているか

- **エネルギー構造の転換**：都市ガス小売全面自由化による競争激化に加え、2050年カーボンニュートラルに向けた低炭素化（e-メタン・水素等）が急務。
- **人的資本開示の圧力**：2023年からの情報開示義務化を受け、投資家からの関心が高まっている。経産省調査では「投資対効果の把握」が日本企業の最大のボトルネック（48.9%が未検討）。特に電気・ガス業界の調査参加率は0.4%と極めて低く、業界内に先行事例がほぼ存在しない。

### ⑤ 今回の企業テーマの背景

1. **投資への説明責任**  
   2025年度の人事制度改定に伴う人件費増に対し、経営層から「定量的成果を示せ」と求められている。人件費は「確定したコスト」として既に発生しており、成果を示せなければ人的資本投資が単なるコスト管理に逆戻りするリスクがある。

2. **インパクトパスの数値化**  
   人的資本レポート2025で「投資→成長→企業価値」の循環を描いたが、各矢印に係数がない。1P＝約10万円はその最初の近似係数の試み。ただし経産省調査では「投資対効果の把握」は全項目中最低スコア（2.63/6.0）であり、この課題は西部ガスHD固有の問題ではなく日本企業全体が超えられていない「測定の壁」でもある。

3. **測定インフラと体制が2025年に初めて揃った**  
   エンゲージメント調査・人的資本委員会新設・人事制度改定・人的資本レポート初発行が全て今年度に集中。「やれる状態」になった。ただし経産省調査では非管理職層の進捗実感スコアが2.1と低く、経営層（3.6）との断層（Fault Line）が存在している。体制は整ったが現場への浸透はこれからという状態。

4. **2040年に向けた構造的な人材不足リスク**  
   経産省「2040年の就業構造推計（改訂版）」（2026年3月）によると、九州ブロックでは2040年に-31万人の需給ミスマッチが見込まれており、特に現場人材・専門職の不足が顕著。エネルギーインフラ業は大規模投資が想定される産業類型であり、保安人財をはじめとする現場人材の確保・育成は待ったなしの課題。「今動かさなければ手遅れになる」という外部環境の緊迫感がクエストの背景にある。

## 課題④：前提・イシューツリー・論点・不足

### 「人的資本投資の効果を、経営の言葉で証明せよ」の分解

- **「人的資本投資」とは何か？** → 人の採用・育成・配置・環境整備にかけるあらゆるリソース（金・時間・仕組み）
- **「効果」とは何か？** → 投資によって生まれた変化・成果
- **「経営の言葉」とは何か？** → 経営層が意思決定に使える言葉

**【参考：人的資本開示のグローバル基準（4要素）】**（ISO30414や人材版伊藤レポート）  
A ガバナンス：誰が人材戦略を監督するか／B 戦略：事業と人材戦略の連動／C リスク管理：人材リスクの特定・評価／D 指標と目標：何を測り何を目指すか  

本クエストでは経営層への証明という観点から **B-D** を以下の3つの言葉に再構成。（A：ガバナンスは原因分析で別途扱う）

- **① 財務の言葉**：ROI・ROE・ROICなど 数字が必須。「いくら投資していくら返ってきたか」
- **② 戦略の言葉**：中計KPI・人財ポートフォリオの充足など 数字でも定性でも可。「この投資が経営目標の達成にどう貢献しているか」
- **③ リスクの言葉**：離職率・機会損失・事業継続リスクなど。「投資しなかった場合に何が失われるか」

**「証明する」とは何か？** → 現場の言葉を、上記3つのいずれかに変換し、経営層が意思決定できる状態にすること。（※厳密な測定がなくても、ロジックとストーリーで証明できる場合がある。）

整理すると、この問いは3つに分解できる。  
**証明①：財務の言葉で証明する**／**証明②：戦略の言葉で証明する**／**証明③：リスクの言葉で証明する**

切り口は **軸A：データ・指標は存在するか（現状把握）**／**軸B：投資との因果関係が見えるか（因果）**／**軸C：経営判断に使える形になっているか（活用）**

---

### 【前提の確認：3つのレンズで疑った】

- **消失レンズ**：「証明しなければならない」という前提が消えたら？ → 対話・定性評価での代替、開示義務の最低限対応という選択肢を検討した。ただし経営会議での説明責任・投資家対応・中計KPIの存在を踏まえると現実的にとれない。→「何のための証明か」を問い直す必要がある。
- **誰得レンズ**：「証明できない」ままで誰が得しているか？ → 人事部・管理職には「証明責任を負わなくて済む」構造的な利益がある。経営層は困っているが手段を持っていない。→ 誰が証明の主体になるべきか。
- **逆転レンズ**：「会社が証明する」を逆転すると？ → 証明の主体・方法・受け手が全て変わりうる。数字か、ストーリーか。上からか、現場からか。→ どんな設計が有効か。

---

### イシューツリー（要約）

- **【証明①】財務の言葉で証明できない**  
  指標と目標：A.データ・指標の現状（ROIは算出済みだが施策レベルが紐づいていない）／B.因果関係の不在（どの施策がROIを動かしたか不明、1P＝約10万円の算出ロジック未確立）／C.経営判断への非接続（ROI→ROE・ROICの変換ロジックが存在しない、目標値・外部ベンチマーク未設定）

- **【証明②】戦略の言葉で証明できない**  
  戦略：A.活動量KPIは存在するが「何をしたか」の記録にとどまる／B.各施策が中計KPI達成にどう貢献したか不明、SBU別の戦略人財要件とポイント基準が未連動／C.経営戦略と人財施策の連動が方針レベルにとどまり数式・ロジックになっていない

- **【証明③】リスクの言葉で証明できない**  
  リスク管理：A.アブセンティーイズム・プレゼンティーイズム・離職率は存在するが単年度の数字にとどまる／B.投資によってこれらの指標がどれだけ改善したか不明、「投資しなければどうなっていたか」という反事実が語れない／C.若年保安人財の不足リスクは認識されているが「何年後に何人不足しいくらの機会損失が生じるか」未試算

- **【証明④】ガバナンスとして機能していない**  
  ガバナンス：A.人的資本委員会は2025年に新設されたが審議・フィードバックのサイクルが未確立／B.先行指標と遅行指標のタイムラグを考慮したモニタリング設計が未整備／C.「どんなデータがあれば投資の追加・撤退を判断できるか」という意思決定基準が未定義、証明の主体と責任が不明確

---

### 構造整理：A・B・C・D

- **A. データの問題**  
  A1. そもそも存在しないデータがある（施策レベルの投資額が未記録、1on1・OJTの実施内容・時間が未収集、各施策が誰にどう届いたか不明）／A2. データはあるが孤立している（ROI・エンゲージメント・挑戦指数・プレゼンティーイズムは存在するが相互に紐づいておらずどの投資がどの指標を動かしたか追えない状態）

- **B. 因果連鎖が存在しない**  
  インパクトパスの矢印に係数がない状態。先行・遅行指標の時間軸設計がない。

- **C. 経営判断に使う設計になっていない**  
  傾向把握止まり。目標値・ベンチマーク未設定。人的資本委員会の意思決定基準が未定義。

- **D. 現場の運用設計が未整備**  
  D1. 申告側：現場が「なぜやるのか」の納得感を持てていない／D2. 承認側：承認基準が統一されていない→人によって評価が変わる、承認負荷が設計されていない→部門長が全件見るのは無理／D3. データの信頼性：承認基準がバラつくと集まったデータが信頼できる数字にならない

**A（データ）が解決しないと B（因果）が見えない。A・Bが解決しないと C（経営判断）に使えない。A・B・Cが解決しても D（現場）が動かないとそもそもデータが集まらない。つまり D→A→B→C の順で解決しないと証明できない。**

---

### 論点（3つ）

1. **論点① なんのために証明するのか**  
   → 経営会議承認・現場変革・投資家開示・業界展開のどれが本当の目的か → これで設計の方向が全部変わる → 全ての前提

2. **論点② 業界展開を見据えているか**  
   → 西部ガス固有の設計か汎用性が必要な設計か → プロダクトのスコープが変わる

3. **論点③ 1P＝約10万円のロジックは誰がどう決めたか**  
   → 因果連鎖の唯一の係数 → 経営層は納得しているか → これが崩れると証明全体が崩れる

### 不足（3つ）

1. **不足① 現場への説明をどう設計しているか**  
   → 現場が動く理由が設計されていないとデータが集まらない → 「やらされ感」なのか「意義を感じているか」か外から見えない

2. **不足② 部門長の日常業務の実態が不明**  
   → 承認にかけられる時間／使うデバイス・既存ツール／承認の判断基準の有無 → これが分からないと誰も使わないプロダクトになる

3. **不足③ 既存システムに何のデータがあるか**  
   → タレントパレット等で何が収集済みか → ゼロから作るのか既存データを繋ぐのかで設計が全く変わる

## 仮説（課題⑤）

**【仮説の前提】** イシューツリーの根本原因は **D→A→B→C** の順で解決が必要。

---

### 仮説1：D：承認基準が統一されないと、集まったデータは経営判断に使えない

| 項目 | 内容 |
|------|------|
| **仮説本文** | もし部門長間で承認基準が統一されず「A部門では通るがB部門では却下」という状態が生まれるなら、データは集まっても信頼できる数字にはならず、証明の土台が崩れる。 |
| **誰についての仮説か** | 部門長（承認する側）・人財戦略部 |
| **この仮説を置く理由** | 1P申請モデルは現場の申告を部門長が承認する設計。しかし承認基準が属人的になると、同じ行動でも部門によって評価が変わる。さらに部門長が全件承認するのは負荷として現実的でない可能性がある。結局「人が評価する」構造は従来の人事評価と変わらず、データの信頼性が担保できない。 |
| **正しければ起こること** | 承認基準の設計（カテゴリ定義・点数設定・承認負荷の上限）がMVPの最重要設計要件になる。承認基準を事前に決めてしまい、部門長の判断余地を減らす設計が優先される。 |
| **誤っていれば起こること** | 部門長は自然と一貫した基準で承認しており、カテゴリ設計よりUIの使いやすさの方が重要になる。 |
| **何を確認できれば前進・修正できるか** | 「ポイントのカテゴリと基準は誰が事前に定義するか」／「部門長が1日に承認できる件数の上限はどれくらいか」／「全件承認・サンプル承認・自動承認のどれを想定しているか」 |

---

### 仮説2：A：開示してもらえないデータがあることで、係数設計の精度に限界が生じる

| 項目 | 内容 |
|------|------|
| **仮説本文** | もし係数設計に必要なデータ（施策別コスト・部門別収益等）が機密情報として開示されないなら、外部公開データとヒアリングで得られる情報だけで係数の近似値を設計しなければならず、その精度と説得力に限界が生じる。 |
| **誰についての仮説か** | 人財戦略部（情報開示の判断者） |
| **この仮説を置く理由** | 係数設計の最も説得力のある方法はROICツリーへの紐付けや案件別収益貢献だが、これらは社内機密データを必要とする。我々が見られないデータがある場合、代替コスト（外部研修相場）などの外部公開情報を基準にした近似設計に限定される。設計の精度は開示してもらえるデータの範囲に直接依存する。 |
| **正しければ起こること** | 外部公開データと代替コストを基準にした係数設計に限定され、精度の限界を推進シナリオの中で正直に説明する必要がある。PDCAで精緻化していく前提を経営層に理解してもらうことが重要になる。 |
| **誤っていれば起こること** | 詳細な施策別コストや収益データを開示してもらえ、ROICツリーに直接紐付いた精度の高い係数設計が可能になる。 |
| **何を確認できれば前進・修正できるか** | 「係数設計のために開示してもらえるデータの範囲はどこまでか」／「施策別のコストは集計した形なら見せてもらえるか」／「どの粒度のデータなら開示可能か」 |

---

### 仮説3：B：カテゴリ別の係数設計なしに6,000Pを経営の言葉で説明できない

| 項目 | 内容 |
|------|------|
| **仮説本文** | もし全てのポイントを同一の係数（1P＝10万円）で説明しようとするなら、ソウゾウ大学での新規事業PoCと1on1実施が同じ価値という説明になり、経営層の納得を得られない。 |
| **誰についての仮説か** | 経営層・人財戦略部 |
| **この仮説を置く理由** | 6,000Pを達成するには様々な行動が積み上がる。行動の種類によって経営インパクトは全く異なるにもかかわらず、全部を同一単価で括ると「どんな行動でも10万円」という形骸化が起きる。カテゴリ別に係数を変えることで、どの行動が特に価値が高いかを経営層に示せる。 |
| **正しければ起こること** | SランクからCランクまでカテゴリ別の係数設計が必要になり、そのカテゴリ定義がQ1の戦略骨子の核心アウトプットになる。 |
| **誤っていれば起こること** | 経営層はカテゴリの粒度より「合計値が目標を達成しているか」を重視しており、単一係数で十分に機能する。 |
| **何を確認できれば前進・修正できるか** | 「1P＝10万円はどう算出したか（代替コスト・ROICレバー・機会損失回避のどれか）」／「経営層はカテゴリ別の内訳を見たいか・合計値だけでいいか」／「ポイントのカテゴリ設計は誰が決める予定か」 |

---

### 仮説4：C：経営層が求めているのは完全な証明ではなく、PDCAを回せる納得感のある近似値だ

| 項目 | 内容 |
|------|------|
| **仮説本文** | もし経営層が求めているのが「学術的に正確な証明」ではなく「人件費増に対してこれだけのプラス影響が出ていると言えるロジックと、それを改善し続ける仕組み」なら、5月の経営会議で見せるべきは精度の高い数字ではなく「動く仕組みがある」という事実。 |
| **誰についての仮説か** | 経営層（承認する側） |
| **この仮説を置く理由** | 活動量と財務成果の因果関係は世界中のどの企業も完全には証明できていない。経産省調査でも投資対効果の把握は全項目中最低スコア（2.63）。クエスト資料にも「PDCA前提・最初から完璧は不要」と明記されている。西部ガスHDはACT2027でROICツリーマネジメントを経営の柱に据えており、人的資本をこの枠組みに翻訳することが承認の最短距離と考えられる。 |
| **正しければ起こること** | MVPは完成度より「動く仕組みがある」という事実を示すことが優先される。Q1の戦略骨子でPDCAを回す推進シナリオを描くことがQ2・Q3と同等以上に重要になる。 |
| **誤っていれば起こること** | 経営層は精度の高い数字を求めており、ロジックが甘いと承認されない。Q2の効果試算シートの精度を上げることが最優先になる。 |
| **何を確認できれば前進・修正できるか** | 「5月の経営会議で何を見せれば承認されるか」／「役員が一番気にしているのは数字の精度か・動く仕組みがあるかか」／「係数の精度よりPDCAを回す意思があるかどうかを経営層は見ているか」 |
