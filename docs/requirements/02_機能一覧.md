# 機能一覧（ドラフト）

優先度は `サービス企画資料_整理版.md` の「〜5/20」「それ以降」に準拠。ID は本ドキュメント内の参照用。

---

## 1. 認証・セッション

| ID | 機能名 | 概要 | 優先度 | Figma / 画面 | 備考 |
|----|--------|------|--------|----------------|------|
| AUTH-01 | ログイン | ID・PW 等で認証しセッション開始 | 高 | 共通 | デモは固定ユーザ可。本番は TBD |
| AUTH-02 | ログアウト | セッション終了 | 高 | 共通 | |
| AUTH-03 | セッション維持・タイムアウト | 一定時間後の再認証等 | 中 | — | ポリシーは非機能で定義 |

## 2. ダッシュボード（閲覧・分析）

| ID | 機能名 | 概要 | 優先度 | Figma / 画面 | 備考 |
|----|--------|------|--------|----------------|------|
| DASH-01 | トップ／因果ストーリー表示 | 指標カード・列フロー・タブ等の表示 | 高 | `1:24` 相当 | データは API から取得想定 |
| DASH-02 | タブ切替 | 「すべて／挑戦の指標／安心・安全」等 | 高 | 同上 | ラベルは Figma 準拠 |
| DASH-03 | 検索 | ヘッダー検索エリア | 高 | 同上 | 検索対象・全文か ID か TBD |
| DASH-04 | ドロップダウン選択 | 組織・事業等のコンテキスト選択（例: Dropbox 表示部） | 中 | 同上 | マスタ連携 TBD |
| DASH-05 | フィルタ・ソート | 一覧・カードの絞込・並べ替え | 高 | 同上 | 企画資料「ソート検索／選択」 |
| DASH-06 | カード詳細・外部リンク | カードの指標表示・リンクアイコン | 中 | 同上 | 遷移先 TBD |
| DASH-07 | 経営向けサマリー（将来） | 集約 KPI のみ表示モード等 | 低 | — | ロール別 UI は要設計 |
